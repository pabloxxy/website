Simulations

I enjoy simulating random objects. There is an almost mystical sense to it. In many of the cases, you are going to be (almost surely) the first and only person in history to witness that particular sampled object.

These are coded in JavaScript specifically to work on web browsers. The codes are not perfect but they are made with love. I also tried to write some explanations about the objects being simulated and/or the methods used. You can read those by clicking on the + buttons.

  • Brownian Bridge to Brownian excursion via Vervaat transform
